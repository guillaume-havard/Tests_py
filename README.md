projet Bipbip
-------------

Compte à rebours
Avec la possibilité d'avoir deux autres comptes à rebours qui s'alternent.

>>> bipbip.py 
lance l'application
>>> bipbip.py <time1> <time2> <time_total>
lance l'application en préremplissant les differents temps

Fait avec pygame (son) et tkinter
